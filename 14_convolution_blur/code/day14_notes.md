# Day14：图像卷积操作 \+ 高斯模糊

## 开篇：什么是卷积—图像处理的核心操作

卷积（Convolution）是图像处理最基础、最重要的数学操作。均值滤波、高斯模糊、边缘检测、锐化——本质都是卷积，区别只在于**卷积核不同**。

*理解卷积 = 理解图像滤波的底层原理。之后学的 Sobel、Canny、形态学，全部建立在卷积之上。*

## 一、卷积的直观理解

卷积核（Kernel）是一个小的二维矩阵，在图像上"滑动"：

```Plain Text
图像 (大矩阵)              卷积核 (小矩阵, 如 3×3)
┌─────────────────┐           ┌─────┬─────┬─────┐
│ 45  50  55  60  │           │ -1  │  0  │  1  │
│ 48  52  57  62  │   滑动→   ├─────┼─────┼─────┤
│ 50  55  60  65  │           │ -2  │  0  │  2  │
│ 53  58  63  68  │           ├─────┼─────┼─────┤
└─────────────────┘           │ -1  │  0  │  1  │
                              └─────┴─────┴─────┘
```

### 卷积操作四步走

① 将卷积核的"锚点"对准图像的某个像素

② 卷积核覆盖的每个像素 × 核上对应的权重

③ 把所有乘积加起来

④ 和 = 输出图像对应位置的像素值

### 直观动画

```Plain Text
输入图像                    卷积核 (3×3)          输出图像
┌───┬───┬───┐              ┌───┬───┬───┐        ┌───┬───┬───┐
│ 1 │ 2 │ 3 │              │1/9│1/9│1/9│        │   │   │   │
├───┼───┼───┤  ──卷积──→   ├───┼───┼───┤  ──→   ├───┼───┼───┤
│ 4 │ 5 │ 6 │              │1/9│1/9│1/9│        │   │ 5 │   │
├───┼───┼───┤              ├───┼───┼───┤        ├───┼───┼───┤
│ 7 │ 8 │ 9 │              │1/9│1/9│1/9│        │   │   │   │
└───┴───┴───┘              └───┴───┴───┘        └───┴───┴───┘
```

中心像素 5 的输出 = (1\+2\+3\+4\+5\+6\+7\+8\+9\) / 9 = 5

***平均值****这就是均值滤波的本质：用邻域像素的替代中心像素。*

## 二、`cv.blur()` — 均值滤波（盒式滤波）

### 1\. API 与参数

```python
result = cv.blur(image, (15, 15))
```

|参数|值|含义|
|---|---|---|
|src|image|输入图像|
|ksize|(15, 15\)|卷积核尺寸（宽, 高）|

### 2\. 卷积核的样子

15×15 的均值核 = 每个元素都是 1/(15×15\) = 1/225：

```Plain Text
┌─────────────────┐
│ 1/225  ... 1/225│
│  ...        ... │    ← 15×15 个相等的权重
│ 1/225  ... 1/225│
└─────────────────┘
```

### 3\. 效果与规律

|核尺寸|效果|
|---|---|
|(3, 3\)|轻微模糊，去细小噪点|
|(5, 5\)|中等模糊|
|(7, 7\)|较强模糊|
|(15, 15\)|强烈模糊，边缘严重丢失|

### 4\. 均值滤波的优缺点

|优点|缺点|
|---|---|
|实现简单，速度快|边缘也被模糊（不保留边缘）|
|均匀去噪|核越大，图像越糊|
|适合快节奏预处理|椒盐噪声效果差（需要中值滤波）|

🌲 林业场景：均值滤波可作为快速的图像平滑预处理，但因为它不保留边缘，不适合需要清晰树冠/叶片边界的场景。边缘保留用高斯或双边滤波。

## 三、cv\.GaussianBlur\(\) — 高斯模糊

### 1\. API 与参数

```python
result = cv.GaussianBlur(image, (0, 0), 15)
```

|参数|值|含义|
|---|---|---|
|src|image|输入图像|
|ksize|(0, 0\)|核尺寸设为 0 → 从 sigma 自动计算|
|sigmaX|15|X 方向标准差|

### 2\. ksize 和 sigma 的两种用法

**用法一：指定核尺寸，自动计算 sigma**

```python
result = cv.GaussianBlur(image, (5, 5), 0)
```

核尺寸 = 5×5

sigmaX = 0 → OpenCV 自动计算：sigma = 0\.3×((ksize\-1\)×0\.5\-1\)\+0\.8

**用法二：指定 sigma，自动计算核尺寸（代码中的方式）**

```python
result = cv.GaussianBlur(image, (0, 0), 15)
```

核尺寸 = (0, 0\) → OpenCV 自动计算核大小

sigmaX = 15 → 标准差较大 → 强烈的模糊效果

自动计算的核约为 (2×ceil(3×15\)\+1, \.\.\.\) ≈ 91×91 的巨大核

|方式|何时使用|
|---|---|
|固定核尺寸|你知道要多大的卷积核（如常见 3×3, 5×5）|
|固定 sigma|你想控制模糊的"力度"，不关心具体核大小|

### 3\. 高斯核与均值核的根本区别

```Plain Text
均值核 (3×3)：                     高斯核 (3×3, σ=1)：
┌──────────┐                      ┌───────────────────┐
│ 1/9 1/9 1/9│                    │ 0.075  0.124  0.075│
│ 1/9 1/9 1/9│   权重全相等        │ 0.124  0.204  0.124│  中心权重最大
│ 1/9 1/9 1/9│                    │ 0.075  0.124  0.075│  越远权重越小
└──────────┘                      └───────────────────┘
均值 = 周围像素同等对待              高斯 = 中心像素最重要，邻域权重随距离衰减
```

|维度|均值滤波 cv\.blur\(\)|高斯滤波 cv\.GaussianBlur\(\)|
|---|---|---|
|权重分布|均匀（每个邻域像素权重相同）|正态分布（中心大，边缘小）|
|模糊效果|生硬，有"块状感"|自然、柔和，"镜头虚化感"|
|边缘保留|差，边缘一起糊|略好于均值，但仍会模糊边缘|
|数学本质|算术平均|加权平均（权重 = 高斯函数值）|
|参数|只控制核大小|核大小 \+ sigma（模糊强度）|

### 4\. sigma 参数的含义

sigma 小 (如 1\)：窄而尖 = 中心像素占主导，模糊范围小，边缘相对清晰

sigma 大 (如 15\)：宽而扁 = 大量邻域像素参与模糊，模糊范围大，效果强但边缘丢失多

代码中 sigma=15 是一个非常大的值，模糊效果极其明显。

### 5\. 高斯模糊的核心优势

|场景|为什么用高斯|
|---|---|
|自然地虚化背景|高斯权重模拟了真实镜头的散景效果|
|Canny 边缘检测前|必须用高斯模糊去噪，否则 Canny 会产生大量假边缘|
|图像金字塔构建|高斯模糊是尺度空间的基础|
|去除高斯噪声|该噪声本身也是正态分布，高斯滤波正好应对|

## 四、均值滤波 vs 高斯滤波

|维度|cv\.blur\(\)|cv\.GaussianBlur\(\)|
|---|---|---|
|核权重|全部相等|中心大，边缘小|
|公式|sum\(邻域\) / N|sum\(邻域 × 高斯权重\)|
|视觉效果|生硬模糊|柔和模糊（推荐）|
|边缘保留|❌ 差|❌ 也模糊，但略好|
|参数|(ksize, ksize\)|(ksize, ksize\) \+ sigma|
|速度|快（简单算术）|快（可分离卷积优化）|
|典型用途|快速降噪|边缘检测前的去噪、美颜|

**关键结论**：绝大多数场景优先选高斯模糊。 均值滤波的唯一优势是稍微快一点，但视觉效果高斯模糊好得多。除非计算资源极度受限，否则 cv\.GaussianBlur\(\) 是默认选项。

## 五、函数命名解读

```python
def blur_demo():
    result = cv.blur(image, (15, 15))       # 均值滤波
def conv_demo():
    result = cv.GaussianBlur(image, (0,0), 15)   # 高斯模糊
```

|函数名|为什么这么命名|说明|
|---|---|---|
|blur\_demo|均值滤波 = "最常见/最基础的模糊"|API 名字就是 blur|
|conv\_demo|conv = convolution（卷积）|高斯模糊 = 图像与高斯核做卷积|

函数名 conv\_demo 在提醒你：高斯模糊的本质也是卷积，只是卷积核从均匀权重换成了高斯权重。所有滤波都是卷积，区别只在核不同。

## 六、自建卷积实战

如果你想彻底理解卷积，可以创建一个自定义核：

```python
# 自定义 5×5 均值核
kernel = np.ones((5, 5), np.float32) / 25
result = cv.filter2D(image, -1, kernel)
# 等价于
result = cv.blur(image, (5, 5))
```

```python
# 自定义锐化核
kernel = np.array([[ 0, -1,  0],
                   [-1,  5, -1],
                   [ 0, -1,  0]])
result = cv.filter2D(image, -1, kernel)
```

`cv.filter2D()` 是 OpenCV 中最底层的卷积 API——你给它任意核，它帮你做卷积。上面所有滤波函数（blur、GaussianBlur、Sobel 等）底层都调它。
