# Day7：图像像素值统计 \+ 几何形状绘制

## 一、图像像素值统计

### 1\. 为什么需要像素统计

|应用场景|说明|
|---|---|
|图像质量评估|判断图片是否过曝（均值偏高）或过暗（均值偏低）|
|特征分析|方差反映图像纹理复杂度，方差大 = 纹理丰富|
|阈值选取|根据均值和方差自动选择二值化阈值|
|变化检测|对比两张图的均值/方差，判断是否有变化|

🌲 **林业应用**：分析植被图像的均值可以判断植被覆盖度（绿色越多，G通道均值越高）；方差可以判断林区纹理复杂度（单一树种 vs 混交林）。

### 2\. `np.max()` / `np.min()` — 像素极值

```python
print(np.max(b1[:, :, 0]))    # B 通道的最大值
print(np.min(b1[:, :, 2]))    # R 通道的最小值
```

|API|作用|示例|
|---|---|---|
|np\.max(img[:, :, ch\]\)|某个通道的最大像素值|B通道最大值|
|np\.min(img[:, :, ch\]\)|某个通道的最小像素值|R通道最小值|

#### 通道索引速查

```python
b1[:, :, 0]   # B 通道（蓝色）
b1[:, :, 1]   # G 通道（绿色）
b1[:, :, 2]   # R 通道（红色）
```

#### 理解极值的意义

|情况|含义|
|---|---|
|max = 255|该通道有像素达到饱和（纯色）|
|max \< 255|没有饱和像素，图像动态范围未充分利用|
|min = 0|有纯黑像素|
|min \> 0|没有纯黑，图像可能偏亮|

### 3\. cv\.meanStdDev\(\) — 均值与标准差（核心！）

```python
means, dev = cv.meanStdDev(b1)
print(means, dev)
```

#### 函数签名

```python
mean, stddev = cv.meanStdDev(src)
```

|返回值|类型|形状|含义|
|---|---|---|---|
|mean|NumPy 数组|(3, 1\)|B、G、R 三个通道的均值|
|stddev|NumPy 数组|(3, 1\)|B、G、R 三个通道的标准差|

#### 解读

```Plain Text
means[0] = B 通道均值
means[1] = G 通道均值
means[2] = R 通道均值
dev[0] = B 通道标准差
dev[1] = G 通道标准差
dev[2] = R 通道标准差
```

#### 均值 vs 标准差

|统计量|反映什么|示例|
|---|---|---|
|均值|整体亮度水平|均值高 = 图片偏亮，均值低 = 图片偏暗|
|标准差|像素值分散程度|标准差大 = 对比度高/纹理丰富，标准差小 = 图片平坦/单一|

一张纯色图片：标准差 ≈ 0；一张有丰富纹理的森林图：标准差很大。

#### 实际应用示例

```python
means, dev = cv.meanStdDev(image)
# 判断是否过曝（均值 > 200 可能过曝）
if means[0] > 200 and means[1] > 200 and means[2] > 200:
    print("图片可能过曝")
# 判断是否模糊/平坦（标准差 < 30 可能缺少纹理）
if dev[0] < 30 and dev[1] < 30:
    print("图片可能模糊或纹理单一")
```

## 二、图像几何形状绘制

### 1\. 应用场景

|场景|绘制内容|示例|
|---|---|---|
|目标检测标注|矩形框 \+ 标签|框出人脸，写"99% face"|
|语义分割可视化|多边形轮廓|圈出树冠区域|
|ROI 标注|矩形/圆形|标记研究区|
|结果展示|文字标注|标注识别结果、概率|

🌲 **林业场景**：在无人机航拍图上框出单株树木、标注树种名称、绘制样方区域边框。

### 2\. `cv.rectangle()` — 绘制矩形

```python
cv.rectangle(b1, (280, 100), (480, 320), (0, 0, 255), 2, 8, 0)
```

#### 参数详解

|参数|值|含义|
|---|---|---|
|img|b1|要绘制的图像（直接在原图上改）|
|pt1|(280, 100\)|矩形左上角坐标 \(x, y\)|
|pt2|(480, 320\)|矩形右下角坐标 \(x, y\)|
|color|(0, 0, 255\)|红色（BGR格式）|
|thickness|2|线条粗细（像素），\-1 表示填充|
|lineType|8|线条类型：8 = 8连通线（默认）|
|shift|0|坐标小数点位数（一般不调）|

#### 注意事项

坐标顺序是 (x, y\)，不是 NumPy 的 [row, col\]！

(280, 100\) → x=280（列），y=100（行）

容易和 ROI 切片 [y1:y2, x1:x2\] 搞混，注意区分

### 3\. `cv.circle()` — 绘制圆形

```python
cv.circle(b1, (200, 200), 100, (255, 0, 0), 2, 8, 0)
```

|参数|含义|
|---|---|
|center|圆心坐标 (x, y\)|
|radius|半径（像素）|
|color|(255, 0, 0\) = 蓝色|
|thickness|2 = 线宽，\-1 = 填充实心圆|

### 4\. `cv.line()` — 绘制直线

```python
cv.line(b1, (100, 100), (300, 300), (0, 255, 0), 2, 8, 0)
```

|参数|含义|
|---|---|
|pt1|起点坐标|
|pt2|终点坐标|
|color|(0, 255, 0\) = 绿色|
|thickness|线宽|

### 5\. `cv.putText()` — 绘制文字（标注核心）

```python
cv.putText(b1, "99% face", (280, 100), cv.FONT_HERSHEY_SIMPLEX, 1.0, (0, 0, 255), 2, 8)
```

#### 参数详解

|参数|值|含义|
|---|---|---|
|img|b1|目标图像|
|text|"99% face"|要显示的文字|
|org|(280, 100\)|文字左下角起点坐标|
|fontFace|cv\.FONT\_HERSHEY\_SIMPLEX|字体类型|
|fontScale|1\.0|字体缩放比例|
|color|(0, 0, 255\)|红色|
|thickness|2|文字粗细|
|lineType|8|线条类型|

#### 常用字体

|字体|风格|
|---|---|
|FONT\_HERSHEY\_SIMPLEX|无衬线（最常用）|
|FONT\_HERSHEY\_PLAIN|小号无衬线|
|FONT\_HERSHEY\_DUPLEX|加粗无衬线|
|FONT\_HERSHEY\_COMPLEX|衬线体|
|FONT\_HERSHEY\_TRIPLEX|粗衬线|
|FONT\_HERSHEY\_SCRIPT\_SIMPLEX|手写体|

#### 文字定位技巧

org 是文字基线左下角位置，不是左上角！

如果文字起点太靠上，文字会跑到图片外面

计算尺寸：`cv.getTextSize()` 可获得文字的 (宽度, 高度\)

### 6\. 绘图函数的共同特点

|特点|说明|
|---|---|
|原地修改|直接在原图上画，不返回新图|
|坐标格式|\(x, y\)，x=列坐标，y=行坐标|
|颜色格式|\(B, G, R\)，不是 RGB|
|thickness = \-1|填充图形（实心）|
|lineType = 8|默认 8 连通线（平滑），也可选 4（锯齿）|

## 三、标注流程标准模板

```python
# 1. 读取图像
img = cv.imread("image.jpg")
# 2. 框出目标
cv.rectangle(img, (x1, y1), (x2, y2), (0, 255, 0), 2)
# 3. 添加文字标签
cv.putText(img, "label", (x1, y1 - 5),
           cv.FONT_HERSHEY_SIMPLEX, 0.8, (0, 255, 0), 2)
# 4. 显示/保存
cv.imshow("result", img)
```

注：y1 \- 5 让文字出现在矩形框上方，避免文字和框重叠。
